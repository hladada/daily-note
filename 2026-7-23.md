# 今日总结

## 1.学会使用 markdownlint插件

- markdownlint插件可以一键帮我修改显而易见的格式问题

## 2.规范书写复选框

1. \[x]:x要小写，并且里面无需加空格

2. \[ ]:空复选框里面必须加上空格

## 3.关于markdown格式写表格要注意

1. 表格每一行的竖线 | 必须严格上下对齐、列宽度规整

2. 行末尾不能存在多余的空格

3. 面对长链接把表格撑得太宽时

- 解决方案：把一长串网页链接换成文字简述并附上注释比如（网页链接[^1]），然后在下面标明[^1]所表示的注释

## 4.通篇统一一种代码写法

- 所有的代码写法统一为围栏写法，消除报错
- 比如

  ```c#
   using System;
   ```
